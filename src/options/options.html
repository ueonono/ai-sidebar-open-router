AI Sidebar — Réglages

100% BYOK : vos clés sont stockées localement dans ce navigateur (storage.local), jamais synchronisées, et envoyées uniquement à l'API que vous choisissez. Aucune clé n'est fournie par l'extension — renseignez la vôtre, ou pointez un modèle local sans clé.

Connexion rapide

Avec un compte : seul OpenRouter permet de se connecter via un compte (Google / GitHub / email) — il débloque alors tous les modèles, y compris des gratuits. C'est le moyen le plus simple de démarrer sans gérer de clés.

Avec vos propres comptes fournisseurs : la plupart offrent un palier gratuit (Gemini, Groq, Mistral, Cerebras, OpenRouter…) — récupérez une clé gratuite via le lien fourni dans chaque carte ci-dessous, puis collez-la. Vous pouvez aussi pointer un modèle local (Ollama / LM Studio) sans aucune clé. Un fournisseur n'apparaît dans le sélecteur de la sidebar qu'une fois connecté.
